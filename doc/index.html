Wenlab server 使用說明

研究室有一臺 server，爲了讓研究室的大家可以有一個可以在雲端進行資料分析的環境，以及使用一些網路服務，而裝了一些可以透過瀏覽器使用的伺服器服務。

Rstudio server

Rstudio server 是讓大家可以直接在瀏覽器上進行 R 的資料分析，或是直接修改 shiny app 的程式。

Jupyterhub

是一個由 ipython 發展而轉生成的資料分析平臺。

Shiny server

是一個 host shiny app 的服務。

FTP

讓大家可以將資料上傳，將結果下載的管道

資料庫

關聯式資料庫

圖資服務

Geoserver/mapserver 類，讓大家可以直接在 Rstudio-server/jupyterhub 上用資料，或是在單機版的 ArcGIS/Qgis 叫圖資。
