關於 Wenlab Server

硬體

  1. 型號：ASUS TS110-E8-PI4 Server
  2. CPU: Intel Xeon CPU E3-1220 v3 @3.10GHz
  3. Ram: 16GB

系統

OS: Debian 8 Jessie (stable)

Server application

  1. RStudio server
  2. Jupyterhub
  3. Shiny server
  4. vsftpd
  5. Apache2
  6. Postgresql + PostGIS
  7. Mysql (未完成設定)
  8. Geoserver/Mapserver/TileStache (規劃中)

  2. 臺灣戰後人口普查 - 互動式網頁地圖版
  3. CDC 計畫
  4. 智慧城市子計畫4
