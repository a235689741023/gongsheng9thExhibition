vsftpd

服務

這臺 Server 的 FTP。

操作

  1. 安裝並打開 filezilla client / netdrive 之類連接 ftp 軟體
  2. Host 設定爲 wenlab.geog.ntu.edu.tw 或 140.112.64.62
  3. protocol: ftp-file transfer protocol
  4. encryption: Use plain FTP
  5. Logon type: Normal
  6. user/password 請用自己的帳號密碼； meeting 與公用 FTP資料帳號密碼請恰研究室
