Rstudio Server

服務

Rstudio server 是 rstudio 的雲端版。在瀏覽器上就能針對在雲端的資料進行計算分析，以及資料處理。

操作

  1. 連上 http://wenlab.geog.ntu.edu.tw:18000
  2. 以自己的帳號密碼登錄，若有問題請聯絡研究室相關人員
  3. 登錄後的界面與單機版 rstudio 基本上一樣
  4. 理論上可以安裝 package，但只能在自己的帳號應用，在 shiny app 無法應用。若有需要安裝 package，也同樣請聯絡研究室
